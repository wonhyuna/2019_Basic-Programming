# 회원가입 양식 만들기

1. 아이디, 비밀번호, 비밀번호 확인, 이름, 성별, 전화번호, 이메일, 자기소개 작성란이 존재
2. 비밀번호는 보이지 않게 password 형식을 사용
3. 취소 버튼을 누르면 작성한 것이 사라짐
4. 회원가입 완료 버튼을 누르면 완료가 됨에 동시에 작성한 것이 사라짐
